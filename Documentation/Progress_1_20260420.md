## Dataset findings

### FashionIQ — verified by hands-on download

- Triplet schema: `{"candidate": ASIN, "target": ASIN, "captions": ["<mod1>", "<mod2>"]}` — two short relative modifications per triplet.
- Val sizes: **dress 2,017 / shirt 2,038 / toptee 1,961** triplets; galleries 3,817 / 6,346 / 5,373.
- ASIN→URL mappings live in a separate companion repo because FashionIQ was released at ICCV 2019 and the authors didn't redistribute Amazon product images themselves. FashionIQ README line 9 explicitly points at `hongwang600/fashion-iq-metadata` for images.
- Amazon URLs still resolve in 2026 — 28/28 fetches succeeded in the notebook run.
- License: CDLA-Permissive — research-use clean.
- Standard metric: Recall@10 + Recall@50 averaged across the three categories.
- TTS-friendly: captions are terse phrases like `"is shiny and silver with shorter sleeves"`, `"is solid white"`.

### FACap — verified by reading the committed JSONs

- Triplet schema: `{"candidate": <path>, "target": <path>, "captions": ["<mod>"]}` — one detailed modification per triplet. Paths reference `f200k_images/...` from Fashion200k.
- Train sizes per category: **dress 59,082 / jacket ~24k / outfit ~32k / pants ~29k / skirt ~34k / top ~30k** — roughly 200k+ triplets total.
- Each category also ships `*_train_captions.json` — a dict mapping image path → one ~80-word fine-grained description. That's the target-side supervision Method A needs.
- Modifications are longer (~23 words) and more descriptive than FashionIQ's — good for fine-grained retrieval but stiffer when read aloud.
- To actually visualize FACap triplets with real images, still need to download **Fashion200k** + **DeepFashion-MultiModal** separately (not done today).
- License: not stated in the repo or project page — open question for the mentor.

### enhFashionIQ — bonus

FACap's refined FashionIQ validation set (`data/fashioniq/enhfashioniq_triplets/`) uses the **same ASIN keys** as FashionIQ val. So once the 28 FashionIQ images were downloaded, the notebook could overlay enhFashionIQ's longer refined caption next to the original two short captions on the same images. Sizes: 3,817 dress / 6,346 shirt / 5,373 toptee refined triplets.
